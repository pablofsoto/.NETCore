# Projects Built in .NET Core

## Projects

- Vega App:  Vehicle Dealer Application 
    (Single Page Application Built with ASP.net Core 2 and Angular 6)

	### To run the project: 
	```
	$ npm install
	$ dotnet restore
	$ dotnet ef database update
	$ dotnet watch run 
	```
